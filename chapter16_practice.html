2 Task 1: Quick recall

The three core ggplot2() steps are:

  • supply a data frame: First we have to tell ggplot() what the dataframe is that we intend to plot. That will kind of set up the blank canvas that initiates the plotting process. It will also let ggplot() know where to look later for the following steps.
  • map aesthetics: Next we use aes() to map the aesthic parts of the graph, although this is kind of an overstatement because really what we’re doing here is letting it know what to use as the x and y values. Using + we can specify different layers of the graph which each have their own aes() call.
  • add geometric objects: Finally, after establishing the x and y values, we use geom_*() functions to put the data onto the graph, e.g., through geom_point to make a scatter plot.

3 Task 2: Scatter plot

Goal: Create a scatterplot from the built-in iris dataset, and save it in an object called Pretty_Flowers. Put sepal length on the horizontal axis and petal length on the vertical axis. Color the points by species so that each species has a different color. Make the points moderately large so they are easy to see. Finally, label the x-axis as “Sepal Length (cm)” and the y-axis as “Petal Length (cm)”.

Pretty_Flowers <- iris %>%
  ggplot(aes(x = Sepal.Length, y = Petal.Length, color = Species))+
  geom_point(size = 3)+
  xlab("Sepal Length (cm)")+
  ylab("Petal Length (cm)")

5 Task 4: Density plot

Goal: Create a density plot using the built-in CO2 dataset and store the plot in an object called CO2_Graph. Put CO2 uptake on the horizontal axis. Use plant type to define separate smooth curves, so each plant type has its own density curve filled with a different color. Make the filled areas partially transparent so the curves can overlap and still be visible where they intersect. Finally, add the title “CO2 Uptake by Plant Type” (use ggtitle()) at the top of the plot.

CO2_Graph <- CO2 %>%
  ggplot(aes(x = uptake, fill = Type))+
  geom_density(alpha = 0.5) + 
  ggtitle("CO2 uptake by plant type")

CO2_Graph

Comment: From this graph, we can see that the plants in Mississippi take up less CO2 than those in Quebec. Both distributions are somewhat bimodal with two different “maxima”. The fact that these graphs overlap one another tells me that there are lot of plants in both regions with the same CO2 uptake. This might be because they are the same kind of plants or it might be because the conditions are different. We can’t know which conclusion to draw without more information.

6 Task 5: Histogram

Goal: Create a graph from the heights dataset and save it in an object called Tall_Density. Put height (in inches) on the horizontal axis. Draw a histogram of heights using bars that are 1 inch wide, filled in blue with white borders. Scale the histogram so that the vertical axis shows density (proportion), not raw counts. On top of this, add a smooth density curve that shows the overall shape of the height distribution, using a dark orange line. Fill the area under that density curve with the same dark orange color, but make it partially transparent so you can still see the blue histogram bars underneath. Label the x-axis “Height (inches)” and the y-axis “Density” (Hint: See Layering geoms and use geom_histogram() first, then geom_density().)

Tall_Density <- heights%>%
  ggplot(aes(x = height)) +
  geom_histogram(aes(y = after_stat(density)),
                 binwidth = 1,
                 fill     = "blue",
                 color    = "white") +
  geom_density(color    = "darkorange",
            fill      = "darkorange",
            alpha     = 0.3,
            linewidth     = 1) +
  xlab("Height (inches)") +
  ylab("Density")

7 Task 6: Reflect

When would a density graph be more useful than a histogram? I don’t really know because histograms are for data which can be placed into bins and it might be useful to really see what falls in each bin. However, the bin size does affect how the histogram plot looks, so if you picked an unreasonable bin size, then the density plot might be more useful (let’s test this below). The density plot might also be better for continuous data where all x values are possible.

7.1 Testing the effect of bin size

Tall_Density_narrow <- heights%>%
  ggplot(aes(x = height)) +
  geom_histogram(aes(y = after_stat(density)),
